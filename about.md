# CustomProfiles
Customize your profile with a new and interesting interface!


## **News!**
- Discord Button ![discord button](frame:gj_discordIcon_001.png?scale=0.375) on Socials Menu!
- Clickeable tag `@username` for copy to clipboard!
- Comment lines bug fix! ![check](frame:GJ_completesIcon_001.png?scale=0.375)

## **Features**

- Customization for Corners
- Customization for Colors
- Disable info button
- Profile colors animation!

## **How to customize it**

Customize your profile pressing the button go to the gray options button found in your profile -> ![gray options](frame:accountBtn_settings_001.png?scale=0.375), there you will have all the options to add or remove from the profile.
